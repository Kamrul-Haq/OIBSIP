📖 Project Overview

The **Iris Flower Classification** project is a supervised machine learning application that predicts the species of an Iris flower based on its sepal and petal measurements. It is one of the most popular beginner-friendly classification problems and demonstrates the complete machine learning workflow, from data preprocessing to model evaluation.

The model classifies flowers into the following three species:

* 🌼 **Iris Setosa**
* 🌷 **Iris Versicolor**
* 🌺 **Iris Virginica**

---

## 🎯 Objective

The primary objective of this project is to build a machine learning model that can accurately classify Iris flowers using their physical characteristics. This project helps in understanding data preprocessing, visualization, model training, and performance evaluation.
