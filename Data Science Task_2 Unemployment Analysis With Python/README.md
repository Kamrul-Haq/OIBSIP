# 📊 Task 2: Unemployment Analysis with Python

## 📖 Project Overview

This project was completed as **Task 2** of the **Oasis Infobyte Data Science Internship**. The objective is to analyze unemployment data using Python and gain meaningful insights through **data cleaning, exploratory data analysis (EDA), and visualization**. The project highlights unemployment trends across different regions and helps understand employment patterns using graphical analysis.

---

## 🎯 Objectives

* 📥 Import and explore the unemployment dataset.
* 🧹 Clean and preprocess the data.
* 📊 Perform Exploratory Data Analysis (EDA).
* 📈 Visualize unemployment trends using charts and graphs.
* 💡 Extract meaningful insights from the data.

---

## 🛠️ Technologies Used

* 🐍 Python
* 📊 Pandas
* 🔢 NumPy
* 📈 Matplotlib
* 🎨 Seaborn
* 📓 Jupyter Notebook

---

## 📂 Tasks Performed

* Loaded and explored the dataset.
* Checked missing values and duplicate records.
* Analyzed unemployment rates across different states and time periods.
* Created visualizations including **bar charts, line plots, histograms, box plots, and heatmaps**.
* Interpreted the results and summarized key observations.

---

## 📌 Key Insights

* 📍 Unemployment rates vary across different regions.
* 📈 Employment trends change over time.
* 👥 Labour participation rate influences employment statistics.
* 📊 Visualizations make it easier to identify trends and patterns.
